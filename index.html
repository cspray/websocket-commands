A PHP 7+ micro-framework to help you create Websocket powered applications using amphp/websocket-server

Installation

We recommend using Composer to install this library. 

composer require cspray/websocket-commands

Quick Start

For our quick start we will demonstrate a simple WebsocketCommand that will echo back to the client a value sent in the ClientPayload. Additionally, we will implement a WebsocketCommandMiddleware that will provide a default value if one was not provided by the client.

For a deeper look into the heart of the library check out the Command & Client Payload section.
